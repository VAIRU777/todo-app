##  Features

- Add and view tasks  
- Persistent data storage using PostgreSQL  
- REST API built with FastAPI  
- Fully containerized using Docker  
- Automatic reload for backend during development

- **Frontend:** HTML, JavaScript  
- **Backend:** FastAPI (Python)
